[문자열 폭발](https://www.acmicpc.net/problem/9935)
========

for문 거꾸로 돌릴 때
-------
<pre>
  <code>
    
arr = ['a', 'b', 'c', 'd']

for i in range(3, -1, -1):  
    print(arr[i], end=" ")
# 출력: d c b a    
    
  </code>
</pre>

문자열에 append 기능 쓰고 싶으면?
-------
<pre>
  <code>
s = ""
s += "a"   # 'a'
s += "b"   # 'ab'
print(s)   # "ab"

  </code>
</pre>

문자열을 쪼개려면?
-----
<pre>
  <code>
s = "hello"

#list() 사용
chars = list(s)
print(chars)   # ['h', 'e', 'l', 'l', 'o']
  </code>
</pre>
