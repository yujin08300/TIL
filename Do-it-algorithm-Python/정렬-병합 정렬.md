병합 정렬
=========
> 병합 정렬은 분할 정복 방식을 사용해 데이터를 분할하고 분할한 집합을 정렬하며 합치는 알고리즘이에요.
> 병합 정렬의 시간 복잡도는 O(nlogn)이에요.
> 코딩 테스트에 관련 문제가 자주 출제돼요. 특히 2개의 그룹을 병합하는 원리 꼭 숙지!

왜 nlogn이냐?
-> 8개를 정렬시킬 때, 8번 데이터 접근하는 단계가 총 3(log8)번이므로 8log8 = N*logN이 돼요.
