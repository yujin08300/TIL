구간 합
======
> 구간 합은 합 배열을 이용해 시간 복잡도를 더 줄이기 위해 사용하는 특수한 목적의 알고리즘이에요.
> 코딩 테스트에서 자주 사용해요.

구간 합의 핵심 이론
--------------------
구간 합 알고리즘을 활용하려면 먼저 합 배열을 구해야 해요. 배열 A가 있을 때 합 배열 S는 다음과 같이 정의해요.  

**S[i] = A[0] + A[1] + A[2] + A[3] + ... + A[i-1] + A[i]**
*ex) S[4] = A[0] ~ A[4]의 합*

합 배열 S를 만드는 공식
------------------------
S[i] = s[i-1] + A[i]

**A[2]~A[5] 구간 합을 합 배열로 구하려면?**  
**->S[5]-S[1]**
