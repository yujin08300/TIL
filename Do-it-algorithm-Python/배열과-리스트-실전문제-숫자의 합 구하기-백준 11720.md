숫자의 합 구하기
=================
> 시간 제한 1초 /
> 난이도 브론즈 IV /
> 백준 온라인 저지 11720번

**N개의 숫자가 공백 없이 써 있다. 이 숫자를 모두 합해 출력하는 프로그램을 작성하시오.**

입력
---------
1번째 줄에 숫자의 개수 N(1 <= N <= 100), 2번째 줄에 숫자 N개가 공백 없이 주어진다.

출력
---------
입력으로 주어진 숫자 N개의 합을 출력한다.
